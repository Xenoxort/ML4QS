Project Overview



This project investigates whether sensory data can be used to predict human activities using machine learning techniques. The goal is to analyze data collected from sensors and build classification models capable of recognizing different physical activities performed by individuals.



Research Question



Can sensory data be used to accurately classify and predict human activities?



Objectives

Explore and preprocess sensory data.

Perform exploratory data analysis.

Extract relevant features from sensor measurements.

Train and evaluate machine learning models for activity classification.

Compare model performance and identify the most effective approach.

Dataset



The dataset consists of sensor measurements collected during various human activities. These measurements may include accelerometer, gyroscope, and other wearable sensor data.



Methods



The project will apply data preprocessing, feature engineering, and supervised machine learning algorithms such as:



Decision Trees

Random Forests

Support Vector Machines

Neural Networks

Expected Outcome



The expected outcome is a machine learning model capable of accurately predicting human activities from sensory data and providing insights into the usefulness of wearable sensor measurements for activity recognition.
